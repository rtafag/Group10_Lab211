Thách thức: Xử lý giỏ hàng, tính toán khuyến mãi chồng chéo, logic tồn kho biến thể.

Dựa trên các yêu cầu từ nguồn tài liệu, việc tạo 10.000 dòng dữ liệu không nên thực hiện thủ công mà nên sử dụng script Java hoặc công cụ AI để giả lập dữ liệu từ một "hệ thống cũ" trước khi import vào hệ thống mới

Dưới đây là cấu trúc và mã nguồn Java để bạn tạo ra tệp dữ liệu (dạng CSV, có thể mở bằng Excel) đáp ứng đầy đủ các tiêu chí: 10.000 dòng, có 5-10% dữ liệu lỗi để thử thách hệ thống làm sạch dữ liệu, và bao hàm các thực thể theo yêu cầu

1. Cấu trúc tệp dữ liệu (Excel/CSV)
Tệp dữ liệu của bạn sẽ bao gồm các cột chính để phục vụ mô phỏng E-Commerce:
• User: ID, Username, Phone (chứa lỗi định dạng), Address.
• Shop: ShopID, ShopName.
• Product & Variants: ProductID, Name, Color, Size, Stock (Tồn kho).
• Voucher: VoucherCode, DiscountValue, StackingType (Loại khuyến mãi chồng chéo).
2. Script Java tạo 10.000 dòng dữ liệu mẫu
Bạn có thể chạy đoạn code này trong môi trường Java (NetBeans/Eclipse) đã cài đặt
. Script này sẽ tự động tạo ra các bản ghi với khoảng 5-10% số điện thoại bị thiếu số (lỗi thực tế thường gặp) để hệ thống của bạn thực hiện bước kiểm tra tính đúng đắn khi import
.

import java.io.FileWriter;
import java.io.IOException;
import java.util.Random;

public class DataGenerator {
    public static void main(String[] args) {
        String fileName = "ecommerce_data_10000.csv";
        try (FileWriter writer = new FileWriter(fileName)) {
            // Header cho file Excel/CSV
            writer.append("RecordID,Type,EntityID,Detail,Quantity_Value,Phone_Status\n");

            Random rd = new Random();
            for (int i = 1; i <= 10000; i++) {
                int errorChance = rd.nextInt(100);
                String phone = "090" + (1000000 + rd.nextInt(9000000));
                String status = "Valid";

                // Tạo lỗi ngẫu nhiên 5-10% (ví dụ: số điện thoại thiếu số) [4, 7]
                if (errorChance < 7) { 
                    phone = phone.substring(0, rd.nextInt(5) + 4); // Cắt ngắn số điện thoại
                    status = "Error_InvalidPhone";
                }

                // Giả lập các thực thể khác nhau
                if (i <= 2000) { // 2000 Users
                    writer.append(i + ",User,U" + i + ",User_Name_" + i + ",0," + phone + "\n");
                } else if (i <= 3000) { // 1000 Shops
                    writer.append(i + ",Shop,S" + i + ",Shop_Name_" + i + ",0,N/A\n");
                } else if (i <= 8000) { // 5000 Product Variants (Size/Color)
                    String variant = (rd.nextBoolean() ? "Red" : "Blue") + "-" + (rd.nextBoolean() ? "L" : "M");
                    writer.append(i + ",Product,P" + i + ",Variant_" + variant + "," + (10 + rd.nextInt(100)) + ",N/A\n");
                } else { // 2000 Vouchers
                    writer.append(i + ",Voucher,V" + i + ",DISCOUNT_" + rd.nextInt(50) + "%,0,N/A\n");
                }
            }
            System.out.println("Đã tạo thành công 10.000 dòng dữ liệu tại: " + fileName);
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

• Logic tồn kho biến thể & Flash Sale: Khi thực hiện Simulator real-time, bạn cần sử dụng biến Stock từ dữ liệu trên. Trong kịch bản Flash Sale, hệ thống phải sử dụng cơ chế synchronized trong Java hoặc khóa trong Database để đảm bảo khi hàng nghìn User đặt hàng cùng lúc, số lượng tồn kho không bị âm

• Khuyến mãi chồng chéo: Bạn nên phân loại Voucher thành các nhóm (ví dụ: Voucher của Shop và Voucher của sàn Shopee). Logic tính toán trong code Java phải cho phép áp dụng cả hai nếu thuộc tính isStackable là true

Tạo một đoạn code java để tạo file excel dữ diệu 10000 dòng về mô hình shoppe
Dựa trên yêu cầu của bạn và các quy định kỹ thuật từ các nguồn tài liệu, dưới đây là đoạn mã Java để tạo tệp dữ liệu mô phỏng mô hình Shopee với quy mô ít nhất 10.000 dòng dữ liệu
,
.
1. Phân tích yêu cầu kỹ thuật
• Khối lượng: Hệ thống phải xử lý tối thiểu 10.000 records để đảm bảo tính thực tế
,

• Chất lượng dữ liệu: Bạn bắt buộc phải tạo ra khoảng 5% đến 10% dữ liệu lỗi (ví dụ: số điện thoại thiếu số) để kiểm tra khả năng làm sạch dữ liệu của hệ thống mới khi thực hiện chức năng Import
,
.
• Mục đích: Giả lập dữ liệu từ một "hệ thống cũ" để import vào hệ thống mới mà bạn đang xây dựng
,

2. Đoạn code Java tạo dữ liệu mẫu (định dạng CSV/Excel)
Đoạn code này sẽ tạo ra một tệp .csv (có thể mở bằng Excel) bao gồm các thông tin về sản phẩm, người dùng và đơn hàng cho mô hình Shopee.

import java.io.FileWriter;
import java.io.IOException;
import java.util.Random;

public class ShopeeDataGenerator {
    public static void main(String[] args) {
        String fileName = "shopee_data_10000.csv";
        int totalRecords = 10000;
        Random rd = new Random();

        try (FileWriter writer = new FileWriter(fileName)) {
            // Header của file dữ liệu
            writer.append("RecordID,EntityType,Name,Detail,Price_Quantity,Phone_Status\n");

            for (int i = 1; i <= totalRecords; i++) {
                String type;
                String phone = "09" + (10000000 + rd.nextInt(90000000)); // Số điện thoại chuẩn
                String status = "Valid";

                // Giả lập 5-10% dữ liệu lỗi (số điện thoại thiếu số) [4]
                if (rd.nextInt(100) < 7) { 
                    phone = phone.substring(0, rd.nextInt(5) + 4); 
                    status = "Error_MissingDigits";
                }

                // Phân bổ các thực thể trong mô hình Shopee
                int randType = rd.nextInt(3);
                if (randType == 0) {
                    type = "Product";
                    writer.append(i + "," + type + ",Product_" + i + ",Color-Size-Variant," + (10000 + rd.nextInt(1000000)) + ",N/A\n");
                } else if (randType == 1) {
                    type = "User";
                    writer.append(i + "," + type + ",User_" + i + ",Address_District_" + rd.nextInt(10) + ",0," + phone + "_" + status + "\n");
                } else {
                    type = "Order";
                    writer.append(i + "," + type + ",OrderID_" + (1000 + i) + ",Voucher_Applied_" + rd.nextBoolean() + "," + (rd.nextInt(5) + 1) + ",N/A\n");
                }
            }
            System.out.println("Đã tạo thành công " + totalRecords + " dòng dữ liệu tại: " + fileName);
        } catch (IOException e) {
            System.err.println("Lỗi khi tạo file: " + e.getMessage());
        }
    }
}
